# 授業時間中におけるPTA活動について（GitHub Pages用）

## 配置

このZIPを展開し、`class-time-pta` フォルダを GitHub Pages の公開リポジトリにアップロードしてください。

公開後の想定URL：

```text
https://<ユーザー名>.github.io/<リポジトリ名>/class-time-pta/
```

独自ドメインを使っている場合は、公開サイトのルート配下に `class-time-pta/` ができます。

```text
https://<ドメイン>/class-time-pta/
```

## 同梱ファイル

- `class-time-pta/index.html`：本文ページ
- `class-time-pta/assets/*.jpg`：一次資料の当該箇所画像
- `class-time-pta/assets/jugyo-jikan-pta.pdf`：PDF版資料

## 変更方針

カード型ポータルではなく、本文を読ませる行政資料・論考ページとして構成しています。スクリーンショットとURLは本文の補助根拠として配置しています。

## 注意

既存サイトのトップページや既存CSSは変更しません。既存メニューへ追加する場合は、既存のナビゲーションから `/class-time-pta/` へリンクを1本追加してください。
